# sinav_takvimi_olusturucu
Dinamik Sınav Takvimi Oluşturma Sistemi, üniversitelerde sınav programı oluşturma sürecini otomatikleştiren masaüstü bir uygulamadır. Derslik, ders ve öğrenci listelerini Excelden içe aktarır; kısıtları dikkate alarak çakışmasız bir sınav takvimi üretir ve oturma planlarını görselleştirip PDF’e aktarır. PySide6+ SQLite mimarisiyle geliştirilmiştir.
